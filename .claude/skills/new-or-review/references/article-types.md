# 文章类型模板

## 技术深潜

- 开头：一个具体问题或观察
- 中间：原理剖析 + 代码或数据验证
- 结尾：适用边界，或者自己后来真正学到的一点

## 系统设计

- 开头：业务场景或面试题，直接抛问题
- 中间：角色划分、选型推导、关键取舍
- 结尾：与其他方案的差异，或者你现在更认可的判断

## 读书笔记

- 开头：为什么读这本书，带着什么问题读
- 中间：核心观点 + 自己的理解和批注
- 结尾：对当前项目的启发，或者不同意作者的地方

## 实践复盘

- 开头：要解决什么问题
- 中间：`V1 -> V2 -> V3` 的迭代过程，每版尽量贴数据
- 结尾：最终方案、值得走的弯路、仍遗留的问题，或者自己后来真正想明白的一点
