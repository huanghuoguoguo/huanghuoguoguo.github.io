# 快速检查清单与评分标准

## 快速检查清单

在交付文本前进行以下检查：

### 结构检查

- [ ] 连续三个句子长度相同？打断其中一个
- [ ] 段落以简洁的单行结尾？变换结尾方式
- [ ] 三段式列举？改为两项或四项

### 连接词检查

- [ ] 使用了"此外""然而"等连接词？考虑删除
- [ ] 存在模板化过渡句？删掉或改成自然衔接
- [ ] 揭示前有破折号？删除它

### 内容检查

- [ ] 解释隐喻或比喻？相信读者能理解
- [ ] 出现空洞修饰词？删掉
- [ ] 存在模糊归因？注明具体来源或删掉

### 元话语检查

- [ ] 出现"修改后""改写后""可以改成""如果你愿意"等？必须删除
- [ ] 看起来像聊天回复、润色建议、答疑说明？必须改回正文
- [ ] 有候选版本列表？只输出一个版本

### 保真检查

- [ ] 原文核心意思发生变化？必须回退重写
- [ ] 原文事实、论点、结论发生变化？必须回退重写
- [ ] 标题、编号、章节层级、段落顺序被改动？必须恢复

### 欧式中文检查

- [ ] 存在"进行+名词""开展+名词""作出+名词"组合？替换为具体动词
- [ ] 存在超过10字的长定语？考虑拆分
- [ ] 句子重心在名词而非动词？调整结构

## 质量评分

对改写后的文本进行1-10分评估，总分70。

| 维度 | 评估标准 | 得分 |
| ---- | ---- | ---- |
| 直接性 | 直接陈述事实还是绕圈宣告？10分：直截了当；1分：充满铺垫 | /10 |
| 节奏 | 句子长度是否变化？10分：长短交错；1分：机械重复 | /10 |
| 信任度 | 是否尊重读者智慧？10分：简洁明了；1分：过度解释 | /10 |
| 真实性 | 听起来像真人说话吗？10分：自然流畅；1分：机械生硬 | /10 |
| 精炼度 | 还有可删减的内容吗？10分：无冗余；1分：大量废话 | /10 |
| 语义保真 | 原文意思、事实、论点、结论是否完整保留？10分：完全保留；1分：明显漂移 | /10 |
| 纯净输出 | 是否彻底避免说明、建议、候选版本、聊天腔？10分：完全纯净；1分：污染明显 | /10 |
| **总分** | | **/70** |

---

## 评分标准

| 分数范围 | 评级 | 说明 |
|---------|------|------|
| 63-70分 | 优秀 | 已去除明显AI痕迹且语义稳定 |
| 49-62分 | 良好 | 仍有改进空间，可进入下一轮或结束 |
| 低于49分 | 不合格 | 需要重新修订当前轮 |

---

## 停止条件

满足任一条即停止自动回环：

- 总分达到63分以上
- 高严重度问题（事实错误、语义漂移）已清空
- 连续两轮改动后评分无显著提升（提升小于5分）
- 修订开始损伤真实细节或语义保真
- 用户明确要求停止

---

## 轮次切换判断

| 情况 | 建议 |
|------|------|
| 存在明显欧式中文（弱动词+动名词、定语堆砌） | 优先完成第1轮 |
| 欧式中文已清除，但仍有AI模式（空洞过渡、三段式、模糊归因） | 进入第2轮 |
| 两轮都已完成，评分仍低于49分 | 交给人工裁决 |
| 两轮都已完成，评分49-62分 | 可结束或人工润色 |
| 两轮都已完成，评分63分以上 | 结束 |
