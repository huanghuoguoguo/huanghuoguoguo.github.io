# 新环境开发工具配置清单

## Claude Code 相关

### claude-hud (状态栏增强)

GitHub: https://github.com/jarrodwatts/claude-hud

```bash
# Step 1: 添加市场
/plugin marketplace add jarrodwatts/claude-hud

# Step 2: 安装插件 (Linux 用户先查看文档)
/plugin install claude-hud

# 重载插件
/reload-plugins

# Step 3: 配置状态栏
/claude-hud:setup

# 完成后重启 Claude Code
```

### tlive (远程操作 Claude Code)

GitHub: https://github.com/huanghuoguoguo/tlive

从 Telegram/Discord/飞书发送任务，实时查看进度，远程审批权限。

```bash
# 安装
curl -fsSL https://raw.githubusercontent.com/huanghuoguoguo/tlive/main/install.sh | bash

# 配置（交互式引导）
tlive setup

# 注册 Claude Code 集成
tlive install skills

# 启动守护进程
tlive start
```

安装后在 Claude Code 中执行 `/tlive` 即可从手机发消息给 Claude。

### cc-switch (切换 Claude Code Provider/MCP)

GitHub: https://github.com/SaladDay/cc-switch-cli

支持多应用：Claude Code、Codex、Gemini、OpenCode、OpenClaw。

```bash
# 快速安装
curl -fsSL https://github.com/SaladDay/cc-switch-cli/releases/latest/download/install.sh | bash

# macOS 手动安装
curl -LO https://github.com/saladday/cc-switch-cli/releases/latest/download/cc-switch-cli-darwin-universal.tar.gz
tar -xzf cc-switch-cli-darwin-universal.tar.gz
chmod +x cc-switch
sudo mv cc-switch /usr/local/bin/

# Linux 手动安装
curl -LO https://github.com/saladday/cc-switch-cli/releases/latest/download/cc-switch-cli-linux-x64-musl.tar.gz
tar -xzf cc-switch-cli-linux-x64-musl.tar.gz
chmod +x cc-switch
sudo mv cc-switch /usr/local/bin/
```

**常用命令**：
```bash
cc-switch                    # 交互模式（推荐）
cc-switch provider list      # 列出所有 provider
cc-switch provider switch <id>  # 切换 provider
cc-switch mcp list           # 列出 MCP 服务器
cc-switch mcp sync           # 同步到配置文件
cc-switch --app claude       # 指定应用（默认）
cc-switch --app codex        # Codex
cc-switch update             # 自更新
```

### Claude Code 配置

#### plan 文件输出位置

在项目根目录创建 `.claude/settings.json`：

```json
{
  "plansDirectory": ".claude/plans"
}
```

全局配置编辑 `~/.claude/settings.json`，使用绝对路径：

```json
{
  "plansDirectory": "/home/用户名/my-plans"
}
```

需要 Claude Code >= 2.1.9 版本支持。

### Skills 安装

#### 官方 Skills

```bash
# 克隆仓库
git clone https://github.com/anthropics/skills.git /tmp/anthropics-skills

# 复制 skills 到全局目录
cp -r /tmp/anthropics-skills/skills/* ~/.claude/skills/

# 清理临时目录
rm -rf /tmp/anthropics-skills
```

#### 自定义 Skills

从本博客仓库复制：

```bash
# 克隆仓库
git clone https://github.com/huanghuoguoguo/huanghuoguoguo.github.io.git /tmp/blog

# 复制 skills
cp -r /tmp/blog/.claude/skills/* ~/.claude/skills/

# 清理
rm -rf /tmp/blog
```

包含的 Skills：
- `author-profile` - 作者画像维护
- `new-or-review` - 博客写作/审阅
- `link-posts` - 文章交叉引用
- `humanizer-zh` - 中文降 AIGC 改写

#### 第三方 Skills

```bash
# web-access
git clone https://github.com/eze-is/web-access.git ~/.claude/skills/web-access

# pua
git clone https://github.com/tanweai/pua.git ~/.claude/skills/pua
```

#### OpenSpec Skills (Spec-Driven Development)

GitHub: https://github.com/chyiiiiiiiiiiii/openspec-skills

Spec 驱动开发框架，包含三个核心技能：
- `/openspec-proposal` - 建立变更提案
- `/openspec-apply` - 实作提案
- `/openspec-archive` - 完成并归档

```bash
# 方法1: 手动安装
git clone https://github.com/chyiiiiiiiiiiii/openspec-skills.git /tmp/openspec
cp -r /tmp/openspec/openspec-proposal /tmp/openspec/openspec-apply /tmp/openspec/openspec-archive ~/.claude/skills/
rm -rf /tmp/openspec

# 方法2: 让 Claude 安装
# 对 Claude 说:
# Clone https://github.com/chyiiiiiiiiiiii/openspec-skills to a temp folder, then copy the three skill folders to .claude/skills/
```

初始化项目目录：
```
Create the openspec directory structure and copy template files
```

会建立：
```
openspec/
├── AGENTS.md      ← AI 指令
├── project.md     ← 专案资讯
├── specs/         ← 规格文件
├── changes/       ← 活跃变更
└── archive/       ← 已完成变更
```

## 快速配置脚本

一键配置脚本（Linux/macOS）：

```bash
#!/bin/bash
set -e

echo "=== 安装 Claude Code Skills ==="

# 官方 skills
git clone https://github.com/anthropics/skills.git /tmp/anthropics-skills
cp -r /tmp/anthropics-skills/skills/* ~/.claude/skills/
rm -rf /tmp/anthropics-skills

# 自定义 skills
git clone https://github.com/huanghuoguoguo/huanghuoguoguo.github.io.git /tmp/blog
cp -r /tmp/blog/.claude/skills/* ~/.claude/skills/
rm -rf /tmp/blog

# 第三方 skills
git clone https://github.com/eze-is/web-access.git ~/.claude/skills/web-access
git clone https://github.com/tanweai/pua.git ~/.claude/skills/pua

# OpenSpec Skills
git clone https://github.com/chyiiiiiiiiiiii/openspec-skills.git /tmp/openspec
cp -r /tmp/openspec/openspec-proposal /tmp/openspec/openspec-apply /tmp/openspec/openspec-archive ~/.claude/skills/
rm -rf /tmp/openspec

echo "=== Skills 安装完成 ==="
ls ~/.claude/skills/
```

保存为 `setup-skills.sh`，执行：

```bash
chmod +x setup-skills.sh
./setup-skills.sh
```
