# 关联已有文章

扫描 `_posts/` 目录，找出与当前文章主题相关的已有博文，建议在合适的位置添加交叉引用链接。

## 规则
- 使用 Jekyll 的 post_url 语法：`{% post_url YYYY-MM-DD-slug %}`
- 只在内容确实相关时才建议添加，不要强行关联
- 优先关联同一技术领域的文章（如数据库相关文章互相引用）
- 输出建议插入的位置和链接文案

## 流程
1. 读取当前文章，提取核心主题和关键词
2. 扫描 `_posts/` 下所有文章的标题和 tags
3. 筛选出主题相关的候选文章
4. 对候选文章快速阅读，确认关联性
5. 输出关联建议：插入位置 + 引用方式 + 理由
